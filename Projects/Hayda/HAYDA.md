# HAYDA - Полный проект

**Статус:** 🟢 MVP РАБОТАЕТ (Ripple + Telegram)  
**Название:** HAYDA ("Поехали" узбекский)

## СУТЬ

**Проблема:** Водители такси теряют 1.5-6% дохода (90-270k сум/месяц) на комиссиях парка

**Решение:** HAYDA - приложение с нулевой комиссией парка + всё в одном месте

**Результат:** Экономия 150-250k сум/месяц для каждого водителя

**ЦА:** 600k водителей такси в Узбекистане

---

## 📱 ПРИЛОЖЕНИЕ (ГОТОВО)

✅ Telegram Mini App (на Ripple)
✅ Интеграция с Яндекс Диспетчерской
✅ Договор как парка-партнера Яндекса
✅ Финтех-сервис (вывод/ввод денег)
✅ Проверка самозанятости

🔄 **В разработке:**
- Интеграция с ГАИ (онбординг + данные)
- Контроль штрафов (проверка + оплата)

---

## 💰 ТРИ ПАКЕТА

### STANDARD (БЕСПЛАТНЫЙ)
- 0% комиссии парка (вместо 1.5-6%)
- Всё в одном приложении
- Вывод на карту (только комиссия вывода)

### PRO (300,000 сум/месяц)
- Реферралы: 1% от дохода привлечённых
- Лимит: 100k с одного водителя/месяц
- Пониженная комиссия вывода
- **Net cost может быть 150k если привлечешь 10 друзей**

### CORPORATE (2% от дохода водителей)
- Для парков-партнеров Яндекса
- Вместо собственной комиссии (3-6%)
- Лимит: 150k с одного водителя/месяц

---

## 🚀 GO-TO-MARKET

**Первые водители:** Прямые вызовы Яндекса (продавец подходит к водителю)  
**CAC:** 5-9k сум (минимальная поездка)  
**Аргумент:** Нет комиссии парка = экономия 150-250k/месяц  
**Масштабирование:** Через реферралы (PRO пакет) = вирусный рост
