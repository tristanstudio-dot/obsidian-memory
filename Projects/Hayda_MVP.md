# Hayda - Реальный MVP (работает)

**Статус:** 🟢 MVP РАБОТАЕТ И ИНТЕГРИРОВАН (Ripple + Telegram)

## ✅ ЧТО ГОТОВО

### Приложение
✅ Telegram Mini App (админ-часть + пользовательское)
✅ Интеграция с Яндекс Диспетчерской (договор партнера)
✅ Сервис вывода/ввода средств (отдельный бот Telegram)
✅ Проверка самозанятости

### Партнёры
✅ **Яндекс Такси** - договор подписан
✅ **Финтех-сервис** - платежи и проверка самозанятости

### В разработке (финальная стадия)
🔄 Интеграция с базами ГАИ (для онбординга)
🔄 Подтягивание данных о водителе и машине
🔄 Контроль штрафов (проверка + оплата со скидкой)

## 💰 ТРИ ПАКЕТА

### 1. STANDARD (БЕСПЛАТНЫЙ)
- Нет комиссии парка (вместо 1.5-6%)
- Всё в одном приложении
- Вывод денег (только комиссия вывода)
- **Цена:** 0 сум/месяц

### 2. PRO (300 000 сум/месяц)
- Реферральная система (1% от дохода привлеченных)
- **Лимит:** 100,000 сум/месяц с одного водителя
- Пониженная комиссия вывода
- **Доход:** можешь заработать на рефералах больше чем платишь

### 3. CORPORATE (2% от дохода)
- Для парков-партнеров Яндекса
- Вместо собственной комиссии парка используют HAYDA
- **Лимит:** 150,000 сум/месяц с одного водителя

## 🚀 GO-TO-MARKET

### Привлечение первых водителей
1. **Прямой контакт в такси** - продавец подходит к водителю
2. **Стоимость:** 5-9k сум за одного (минимальная поездка)
3. **Аргумент:** Нет комиссии парка + всё удобно
4. **Дальше:** PRO пакет + реферралы = вирусный рост

### Каналы расширения (после первых 100)
- Telegram канал (обновления, советы)
- Instagram (истории водителей)
- Реферральная программа (основной канал)
- WhatsApp (когда будет API)

### Критическая масса (1,000+ водителей)
- Подключить других агрегаторов (Uber, Maxim, Careem)
- HAYDA становится OS для водителя
- Переключение между агрегаторами в одном приложении
