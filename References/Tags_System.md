## КАК ДОБАВИТЬ

1. Открой файл
2. После # Название добавь блок с тегами
3. Сохрани (Ctrl+S)
4. Граф автоматически обновится

## РЕЗУЛЬТАТ

Когда добавишь теги:
1. Открой граф
2. Включи **Group by tag**
3. Включи **Color by tag**
4. Видишь цветной граф где все организовано!
