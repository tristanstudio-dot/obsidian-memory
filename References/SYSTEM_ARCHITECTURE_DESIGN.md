## 🎯 ОСНОВНАЯ КОНЦЕПЦИЯ

Главный узел - **ТЫ (Станислав)** - центр вселенной.

От тебя отходят **ТРИ ГЛАВНЫХ ВЕКТОРА**:

1. **ЛЮДИ** - те кто вокруг тебя (концентрические круги)
2. **МЫСЛИ** - внутренний мир (идеи, мировозрение, взгляды)
3. **ДЕЯТЕЛЬНОСТЬ** - внешний мир (проекты, бизнесы, дела)

Каждый вектор имеет свой **индексный файл** и разветвляется на заметки.

**КЛЮЧЕВОЙ МОМЕНТ:** Когда в одной заметке упоминается что-то из другого вектора - создаётся СВЯЗЬ через вики-ссылки.

## 🎯 КЛЮЧЕВЫЕ ПРАВИЛА

### Правило 1: Индексные файлы - входные точки
- Каждый большой вектор (PEOPLE, THOUGHTS, ACTIVITIES) начинается с Index.md
- Это **обзор** всего в этом векторе
- **Граф показывает структуру через индексы**

### Правило 2: Упоминание = Связь
- Если в файле упоминается человек → вики-ссылка на его профиль
- Если в проекте упоминается идея → ссылка на идею в THOUGHTS
- Если в идее упоминается человек → ссылка на человека

### Правило 3: Разветвление = Иерархия
- Большой проект = папка с несколькими файлами
- Каждый файл может содержать ссылки на подпроекты
- Глубина разветвления не ограничена

### Правило 4: История сохраняется
- Файл в THOUGHTS не удаляется когда становится проектом
- Становится **источником** для проекта
- **Видна полная цепь: мысль → идея → проект**

### Правило 5: Граф показывает связи
- При Depth 1: видно только прямые связи файла
- При Depth 2: видно всю сеть
- Видны все люди, идеи, проекты которые связаны
