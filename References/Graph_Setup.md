# Obsidian Graph - Настройка

## ОТКРЫТЬ ГРАФ

**Ctrl+P → Open graph**

## ОСНОВНЫЕ ФИЛЬТРЫ

### Depth (глубина связей)
- Depth 1: только прямые ссылки
- Depth 2: через один уровень
- Depth 3: максимум обзора
- **Рекомендация:** Depth 2-3

### Filter by tag
- Введи тег (например: hayda)
- Видишь только файлы с этим тегом

### Search
- Ищет по имени файла
- Пример: HAYDA, Projects

### Orphans
- Включай чтобы найти забытые файлы (без ссылок)

## ФИЗИЧЕСКИЕ НАСТРОЙКИ

`
Repel Force: 100 (узлы отталкиваются)
Link Force: 35 (сила линий)
Center Force: 10 (притяжение к центру)
Distance: 300 (расстояние между узлами)
`

## ЦВЕТОВАЯ СХЕМА

Включи **Color by tag** в настройках.

Используй теги:
- #hayda → красный
- #project → синий
- #finance → зелёный
- #marketing → оранжевый

## ИНТЕРАКТИВНЫЕ ДЕЙСТВИЯ

- **Левый клик** → открыть файл
- **Правый клик** → меню опций
- **Прокрутка** → zoom in/out
- **Перетаскивание** → движение по графу

## РЕКОМЕНДУЕМЫЕ НАСТРОЙКИ ДЛЯ HAYDA

`
Depth: 2
Repel Force: 100
Link Force: 35
Center Force: 10
Distance: 300
Group by tag: ON
Color by tag: ON
`

**Result:** Красивый граф где все проекты разных цветов!
